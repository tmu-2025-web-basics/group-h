遊び方

1. まずは鬼の背後をとり、背中から侵入

2. 階段を登って覚悟を決める

3. わずか2秒ほどのスライドを楽しむ

西村陸が行ってきた！
わざわざ東京の果てまで遠征して見てきた。
足立区舎人車人に会うこの鬼
いざ生で見てみるとその迫力には目を見張るものがあった。
しかし、遊ぶ子供は見当たらない
駅からこのに向かう際に幼稚園や小学校(公演の目の前)
あったので、期待をしていたがまさかの0人だった。
いざ滑ってみるとその理由が判明
この遊具はビジュアルに特化したあまり、遊具としての楽しさが
何一つとしてなかったのだ。
地元の子供達も自分と同じような結論に達していた
と思うと感慨深いものがあった。しかし、見た目に関しては素晴らし
いと言わざるを得ないので、生で見たい人はぜひ行ってもらいたい
